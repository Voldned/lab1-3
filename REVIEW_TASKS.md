## 2) Исправление ошибки
**Задача:** исправить выход за границы массива в `DrawLocMaxMin`.

**Почему:** цикл проходит до `i < masSize`, но внутри обращение идет к `masFF[1][i+1]` и `masF[*][i+1]`. На последней итерации (`i == masSize - 1`) происходит чтение за пределами массива.

**Ожидаемое исправление:**
- изменить условие цикла на `i + 1 < masSize` (или `i < masSize - 1`);
- добавить защиту для случаев `masSize < 2`.

**Где найдено:** `MyProject/drawfunctions.cpp`.

## 4) Улучшение тестов
**Задача:** добавить базовые автоматические тесты для вычислительной части (`f`, `CalcDFuncMas`, границы `DrawLocMaxMin`).

**Почему:** сейчас в репозитории нет тестового покрытия, а критичная логика содержит арифметику и индексирование массивов.

**Минимальный набор тестов:**
1. Проверка корректности веток `f(x, i)` для нескольких `i` и контрольных значений `x`.
2. Проверка `CalcDFuncMas` для маленьких массивов (`masSize = 1`, `masSize = 2`, `masSize > 2`).
3. Регрессионный тест на отсутствие выхода за границы в логике локальных экстремумов.

**Где актуально:** `MyProject/drawfunctions.cpp` / новый тестовый проект в решении `MyProject.sln`.
